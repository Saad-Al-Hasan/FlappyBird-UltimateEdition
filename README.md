🐦 Flappy Bird: Ultimate Edition

A modern and feature-rich recreation of the classic Flappy Bird game developed entirely using **HTML5, CSS3, and JavaScript**, featuring multiple themes, unlockable birds, power-ups, achievements, and dynamic gameplay.



🎮 Live Demo
**Play Online:**  
https://saad-al-hasan.github.io/FlappyBird-UltimateEdition/

📂 Source Code
**GitHub Repository:**  
https://github.com/Saad-Al-Hasan/FlappyBird-UltimateEdition



✨ Features:
🐦 Multiple Playable Birds
- Sparrow
- Parrot
- Eagle
- Dragon

Each bird has its own unique appearance and design.



🌎 Multiple Themes
- ☀️ Day
- 🌙 Night
- 🚀 Space
- 🌊 Underwater

Each theme includes:
- Unique visual effects
- Different ambient environments
- Dedicated background music



🛒 Shop System
Players can spend collected coins to unlock:
- New birds
- New themes
- Special content



🪙 Coin System
- Collect coins during gameplay
- Use coins to purchase unlockable content



⚡ Power-Ups
The game features several power-ups:

| Power-Up          | Description |
|-----------------  |-------------|
| 🛡 Shield        | Grants temporary invincibility |
| 🧲 Magnet        | Automatically attracts nearby coins |
| ✖️2 Double Score | Doubles the score gained |
| 🐌 Slow Motion   | Temporarily slows down the game |



🏆 Achievement System
Unlock achievements by completing various in-game challenges and milestones.



📊 Statistics System
Track:
- Highest Score
- Total Games Played
- Total Coins Collected
- Total Distance Traveled
- Achievement Progress



🎵 Dynamic Music System
Each theme contains its own background music:
- Homepage Music
- Day Theme Music
- Night Theme Music
- Space Theme Music
- Underwater Theme Music



🎯 Progressive Difficulty
The game difficulty automatically increases as the player's score increases.



🎮 How To Play

### Controls

| Action         | Control |
|--------------- |---------|
| Fly            | Spacebar / Mouse Click |
| Pause          | p |
| Navigate Menus | Mouse |



Gameplay Rules:
1. Press **SPACE** or click to make the bird fly.
2. Avoid colliding with pipes and obstacles.
3. Collect coins to purchase new content.
4. Collect power-ups for temporary advantages.
5. Try to achieve the highest score possible.
6. Unlock all birds, themes, and achievements.



🛠 Technologies Used

- HTML5 Canvas API
- Local Storage API
